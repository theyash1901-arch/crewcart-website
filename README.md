## 📱 About CrewCart
CrewCart is a campus delivery app built for students at Chandigarh University. Skip the trip to the milk shop or print shop — we deliver straight to your hostel room, fast.

## 🚀 Services
| Service | Price | Delivery Time |
|---|---|---|
| 🥛 Full Cream Milk (500ml) | ₹39 | 5 minutes |
| 📄 B&W Print | ₹4/page | 10 minutes |
| 🎨 Colour Print | ₹8/page | 10 minutes |
| 🚚 Delivery Fee | FREE | Always |

## ✨ Features
- ⚡ 5–10 minute hostel delivery
- 💳 Secure payments via Razorpay (UPI, Cards, Wallets)
- 🏆 Reward points on every order
- 📎 File upload for print orders
- 🌙 Beautiful dark UI with glassmorphism design

## 🛠️ Tech Stack
- **App:** React Native
- **Payments:** Razorpay
- **Backend:** Google Apps Script + Google Sheets
- **Website:** HTML, CSS, JavaScript
- **Hosting:** Vercel
